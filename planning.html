CSS: Estilos Internos y Externos, Selectores CSS

CSS es el lenguaje que usamos para diseñar un documento HTML.

CSS describe cómo se deben mostrar los elementos HTML.

Este tutorial le enseñará a incorporar los estilos internos y externos de CSS, además selectores CSS

para conocer más de este OVI lo invitamos a consultar más en: w3schools.com

Objetivo General

Este OVI "Objetivo Virtual de Información" de CSS pretende enseñar los estilos Internos y Externos, además de selectores CSS

Objetivos Específicos

  • Formas de aplicar CSS
  • Definir estilos CCS en documentos html
  • Declaración de estilos CSS externos y llamarlos desde un documento html

Características

Entre las características de los estilos tenemos que se pueden usar dentro del cuerpo del documento html (Estilos Internos) o los invocándolos desde otro documento de estilos de cascada con la finalidad de tener menos líneas de programación, más orden, permite modificar estos estilos por separado sin afectar el index.htl o páginas secundarias

Los selectores se utilizan para seleccionar todos los elementos de la página
